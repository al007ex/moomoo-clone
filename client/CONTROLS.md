# Game Controls

## Movement
- **W, A, S, D** - Move character
- **Mouse** - Look around
- **Click / Space** - Gather resources / Attack

## Actions
- None. No have

## Menus
- **Insert** - Open/Close UI Settings Menu (Auto Heal and other settings)
- **Enter** - Open/Close chat
- **Escape** - Close all windows

## Building
- **1-9** - Select weapons and items
- **Click** - Place selected item

## UI Settings Menu (Insert Key)

Press **Insert** to open the UI settings menu where you can:
- Toggle Auto Heal
- Configure other game settings
- Customize your experience

Note: Auto Heal is NOT in the main menu Settings tab - use Insert key to access it.
